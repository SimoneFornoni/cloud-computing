AWS

Il cloud computing é una tecnica che permette alle aziende di richiedere risorse o servizi attraverso l'internet, eliminando il bisogno di avere dei server in housing, dispendiosi da mantenere. A differenza dell'housing, piú costoso perché si affitta uno spazio fisico con all'interno presenti le risorse di cui si hanno bisogno, il cloud computing permette invece di pagare in base al tempo di utilizzo dei servizi ed é un sistema facilmente scalabile, ovvero in grado di aumentare o diminuire le prestazioni dei sevizi in base ai bisogni. Gli Amazon Web Services sono una piattaforma di cloud computing che offre diversi servizi cloud tra cui quello utilizzato in questo progetto, ovvero i calolatori EC2. Su di questi ho installato una macchina linux che tramite command line mi permette di interagire con i server in hosting. Specificando le porte di cui si ha bisogno siamo anche in grado di limitare il traffico e controllarlo.
